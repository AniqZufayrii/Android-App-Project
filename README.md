# 💸 SPLIT: Shared Household Management App

[Android]
[Kotlin]
[Jetpack Compose]
## 📌 About The Project
**SPLIT** is a native Android application developed as a Final Year Project (FYP). It is designed to solve the common pain points of living with roommates by providing an automated, transparent, and user-friendly platform to manage shared expenses and household chores. 

Gone are the days of manual calculations and awkward money conversations!

## ✨ Key Features
* **Automated Bill Splitting:** Instantly calculates who owes what, eliminating manual math errors.
* **Real-Time Dashboard:** A unified "Owes You" summary to keep track of personal financial standing.
* **Digital Expense Ledger:** A fully filterable history of past transactions (Utilities, Rent, Groceries).
* **Interactive Chore Tracker:** Assign, track, and complete daily household duties with a simple checkbox system.
* **Modern UI/UX:** Built entirely with Jetpack Compose featuring smooth animations (e.g., Slide to Delete) and a comfortable Dark Theme aesthetic.

## 🛠️ Technology Stack
* **IDE:** Android Studio
* **Language:** Kotlin
* **UI Toolkit:** Jetpack Compose
* **Local Database:** Room Persistence Library (SQLite)
* **Architecture:** MVVM (Model-View-ViewModel)
